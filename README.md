<h3 align="center">USB Hub</h3>

- a modern usb hub with 2x USB-A and 2x Type-C female ports for input and 1x Type-C female for output to pc/laptop.

----

## Features

* 2x USB-A and 2x Type-C female ports for input.
* 1x Type-C female for output to pc/laptop.
* Faster Transfers.
